PAYable Payment Gateway for WooCommerce from Payable Private Limited, Sri Lanka

== Description ==

PAYable Payment Gateway for WooCommerce is a Sri Lankan Payment Gateway Service which allows you to accept payments online from your customers for your products and services via credit and debit cards. This online payment gateway plugin enables reliable online shopping experience for your customers and lets you sell products and services directly from your own website.

You can simply install this plugin to list “PAYable Online” as a payment method on your WooCommerce website in a few minutes and it requires no technical knowledge when setting up. 

This plugin is highly secured against online threats and hackers. It accepts all internationally recognised and acceptable cards.

== Installation ==

1. Download the PAYable Payment Gateway for WooCommerce plugin zip file and extract to the /wp-content/plugins/ directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the WooCommerce -> Settings -> Payments -> select PAYable Online as payment method
4. Use your PAYable Merchant Account details provided by PAYable as described in the screenshots and configure the plugin. Make sure you tick the Sandbox Mode checkbox if you want to test the plugin with your PAYable Sandbox account.
5. You will notice the PAYable Online payment option in your Checkout page.
That’s all. Now you are ready to accept the payments online from your customers using their credit or debit cards.

No. At the moment, we are supporting Visa, Mastercard, Amex, Diners Club and Discovery payments. Other payment types such as CUP and Lanka QR payments will be available very soon.
